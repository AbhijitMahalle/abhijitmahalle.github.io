Computer Vision Projects

Structure from Motion

Reconstructed a 3D scene and simultaneously obtained camera poses from a given set of images using their feature point correspondence and the concepts of epipolar geometry, triangulation and non-linear optimization.

Visual Odometry

Computed trajectory of a camera placed on an autonomous vehicle using the concepts of feature-matching, RANSAC, epipolar geometry and achieved an accuracy of over 90%.

Visual Sensor Fusion

Fused the output of YOLO detections and LIDAR point cloud projections on a video sequence to detect humans, bicycles, and cars and their corresponding distance from the autonomous vehicle.

Stereo Vision

Estimated depth and disparity for image sequences in Middlebury dataset by implementing the concepts of epipolar geometry, rectification, correspondence, camera calibration, and SSD.

AR Tag Detection and Tracking

Decoded custom Aruco marker using corner detection and fast Fourier Transform and superimposed a custom image and a virtual cube on it using projection, calibration, and homography matrices.

Lane Detection and Turn Prediction

Detected straight and curved road lanes using homography, hough lines, polynomial curve fitting, and predicted turn by calculating the radius of curvature for detected lanes mimicking the lane departure system in autonomous vehicles.

Human Position Estimator

Developed a software by Agile Iterative Process that detects and tracks humans using a pre-trained HOG descriptor and SVM detector of OpenCV.

Face Swap

Swapped faces in videos using traditional approaches like Delaunay Triangulation and Thin Plate Spline and Deep Learning approach with Position Map Regression Network.

Panorama Stitching

Stitched images to create a panorama using traditional (Homography estimation using feature points) and deep learning (HomographyNet: Supervised and unsupervised) methods.

PbLite Edge Detection

Detected edges using a simplified version of the probability of boundary detection algorithm.
